# homebrew-carabiner

Homebrew tap for [Carabiner](https://github.com/lvcabral/carabiner) — a live video overlay for streaming device development and QA.

## Install

```bash
brew tap lvcabral/carabiner
brew install --cask carabiner
```

## Upgrade

```bash
brew upgrade --cask carabiner
```

## Uninstall

```bash
brew uninstall --cask carabiner
brew untap lvcabral/carabiner
```
